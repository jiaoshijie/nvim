# REALLY SIMPLE Vim CONFIG

### normal-mode

| Command        | Motion                       |
| -------        | ---                          |
| `<leader>fo`   | 打开vimrc                    |
| `<leader>/`    | 搜索单词                     |
| `<leader>ps`   | 打开语法检查                 |
| `<leader>tt`   | 背景透明                     |
| `<leader>=`    | 格式化代码                   |
| `<leader>xd`   | 切换为16进制模式             |
| `<leader>xr`   | 切换回文本模式               |
| `<leader>eu`   | 切换编码为utf-8              |
| `<leader>eg`   | 切换编码为gbk                |
