# Simple vim config(non-plugin)

### normal-mode

| Command        | Motion                       |
| -------        | ---                          |
| `F2`           | 打开vimrc                    |
| `<leader><cr>` | 取消搜索高亮                 |
| `<leader>/`    | 搜索单词                     |
| `<leader>fm`   | 打开笔记                     |
| `<leader>fc`   | 清除行尾空格和文件尾部的空行 |
| `<leader>o`    | 打开语法检查                 |
| `<leader>tt`   | 背景透明                     |
| `<leader>=`    | 格式化代码                   |
| `<leader>zz`   | 折叠代码                     |
| `<leader>xd`   | 切换为16进制模式             |
| `<leader>xr`   | 切换回文本模式               |
| `<leader>eu`   | 切换编码为utf-8              |
| `<leader>eg`   | 切换编码为gbk                |
