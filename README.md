# Vimrc

## Keyboard Shortcuts
- `<leader>=space`

### command-mode
| Command | Motion |
| ----    | ----   |
| `<C-f>` | Right  |
| `<C-b>` | Left   |

### insert-mode
| Command | Motion          |
| ------- | ---             |
| `<C-j>` | 进入normal-mode |
| `<C-b>` | left            |
| `<C-f>` | right           |

### normal-mode

| Command      | Motion           |
| -------      | ---              |
| `<leader>xd` | 切换为16进制模式 |
| `<leader>xr` | 切换回文本模式   |
| `<leader>eu` | 切换编码为utf-8  |
| `<leader>eg` | 切换编码为gbk    |
| `F2`         | 打开vimrc        |
